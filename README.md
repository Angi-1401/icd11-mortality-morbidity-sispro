# ICD-11 Mortality and Morbidity Report for SISPRO

This Excel/VBA-based tool is designed to analyze clinical and hospital reports generated by SISPRO (Sistema de Información de Salud), the public health information system used in Venezuela. It provides automated insights into regional health conditions by identifying and summarizing the top causes of mortality and morbidity.

## Features

- **Top 25 Causes of Death Analysis**  
  Automatically extracts and ranks the 25 leading underlying causes of death from SISPRO reports using customizable criteria.
- **Interactive Dashboard**  
  Generates a dynamic dashboard to visualize and summarize key health indicators.
- **ICD-11 API Integration**  
  Connects to the official ICD-11 API to retrieve descriptions for each diagnostic code, enhancing report clarity and accuracy.
- **Optimized for Large Datasets**  
  Efficient algorithms handle extensive datasets (20,000+ rows) with high performance.
- **Secure API Access**  
  Implements two-step authentication for secure access to the ICD-11 API.

## Requirements

- **Operating System**: Windows XP or later 
- **Excel Version**: Microsoft Excel 2010 or later  
- **Macro Support**: Macros must be enabled  
- **API Credentials**: A valid `CLIENT_ID` and `CLIENT_SECRET` from the [ICD-11 API portal](https://icd.who.int/icdapi) are required

## Installation

1. Download the repository as a ZIP file and extract it to your desired location. Alternatively, clone the repository using Git:
   ```bash
   git clone https://github.com/Angi-1401/icd11-mortality-morbidity-sispro.git
   ```
2. Open the `ICD11_Mortality_Morbidity_SISPRO.xlsb` file in Microsoft Excel.
3. Access the `ICD11.bas` module in the VBA editor (alt + F11) to input your ICD-11 API credentials.
    ```vba
    Const CLIENT_ID As String = "YOUR_CLIENT_ID_HERE"
    Const CLIENT_SECRET As String = "YOUR_CLIENT_SECRET_HERE"
    ```
4. Save the workbook to retain your credentials.

## Getting Started

1. Open the Excel workbook and enable macros when prompted.
2. Enter your ICD-11 API credentials in the designated configuration section.
3. Load your SISPRO report into the sheet.
4. Run the macro to analyze the data and generate the dashboard.

## Notes

- Ensure internet connectivity for API access.
- The tool is tailored for SISPRO report formats; other data sources are not supported.
